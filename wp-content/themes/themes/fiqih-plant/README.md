# Fiqih Plant Theme

A custom, premium WordPress theme designed for Fiqih Plant - an ornamental and productive plant nursery.

## Features
- **Nature-Inspired Design**: Uses a curated Green/Beige/Brown palette.
- **WooCommerce Ready**: Custom wrappers and styles for shop pages.
- **Responsive**: Mobile-first design.
- **Speed**: Minimalist coding (Vanilla CSS, no huge frameworks).

## Recommended Plugins
To fully enable the functionality requested, please install the following plugins:

1.  **WooCommerce**: Core e-commerce functionality.
2.  **RajaOngkir (WooCommerce Ongkir)**: For automatic shipping calculation in Indonesia.
3.  **Midtrans / Xendit Gateway**: For Indonesian payment methods (QRIS, VA, E-Wallet).
4.  **Contact Form 7** or **WPForms**: For the Contact page form.
5.  **Instagram Feed** (e.g., by Smash Balloon): To sync the Instagram gallery on the home page.

## Installation
1.  Copy the `fiqih-plant` folder to `wp-content/themes/`.
2.  Go to **Appearance > Themes** in WordPress Admin.
3.  Activate "Fiqih Plant".
4.  Install the plugins listed above.

## Development (Docker)
1.  Run `docker-compose up -d` at the project root.
2.  Access `http://localhost:8000`.
3.  Database Host: `db`, User: `exampleuser`, Pass: `examplepassword`.
